# SignalShape Command

Use this as a slash command body or reusable prompt.

Apply SignalShape to the next response.

1. Classify the task: debug, review, implement, explain, status, writing, decision, or handoff.
2. Pick the smallest useful budget: tiny, work, deep, or handoff.
3. Use the matching response shape.
4. Remove assistant filler and generic AI phrasing.
5. Keep exact commands, errors, paths, line numbers, constraints, assumptions, and verification.
6. Lead with the useful part.

If the task is unclear, ask the shortest question that unlocks action.
